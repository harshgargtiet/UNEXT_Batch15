In your JavaScript code, write a function named "changeColor" that changes the background color of a specified element on the page.
The function should accept two parameters: "elementId" (string) and "color" (string).
Inside the function, use the getElementById method to select the element with the specified "elementId".
Set the style.backgroundColor property of the selected element to the specified "color".
Call the "changeColor" function with an element ID of your choice and a color of your choice to see the background color change.
